# Inventory Pluralization and Totals

This program demonstrates how to:
- Use enums and vectors to represent an inventory system
- Print each item in the inventory with correct pluralization based on its count
- Calculate and display the total number of items

## How it works
1. The `Inventory` namespace defines item types and their counts.
2. The program prints each item and its count, using rules to pluralize the item name correctly.
3. It then prints the total number of items in the inventory.

## Example Output
```
You have 1 Health Potion
You have 5 Torches
You have 10 Arrows
You have 16 in total.
```

## Concepts Covered
- Enum indexing
- String pluralization logic
- Summing vector elements
- Output formatting
