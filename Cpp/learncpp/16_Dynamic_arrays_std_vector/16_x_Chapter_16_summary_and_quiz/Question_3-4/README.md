# Find Min and Max in User-Input Array

This program demonstrates how to:
- Read a sequence of integers from the user into a vector (ending with -1, which is not allowed as the first input)
- Find the indices of the minimum and maximum elements in the vector
- Print the array, and display the index and value of both the minimum and maximum elements

## How it works
1. The user is prompted to enter numbers, ending input with -1 (which is not accepted as the first value).
2. The program finds the indices of the minimum and maximum values in the array.
3. The array and the min/max information are printed.

## Example Output
```
Enter numbers to add (use -1 to stop): 5 2 8 1 7 -1
With array ( 5, 2, 8, 1, 7 ):
The min element has index 3 and value 1
The max element has index 2 and value 8
```

## Concepts Covered
- User input validation
- Vectors and iteration
- Finding min/max indices
- Output formatting
