# Question 2: Potion shop

This project implements a text-based potion shop simulation in C++.

## Features
- Random starting gold generation.
- Inventory system for different potion types.
- Purchase validation and balance checking.
- Summary of purchased items and remaining gold.

## File Structure
- `Question.cpp`: Main source code containing the shop logic and classes.
- `Question.exe`: Compiled executable.

## How to Run
1. Compile the code using a C++ compiler.
2. Run the executable.
3. Follow the on-screen prompts to enter your name and buy potions.
