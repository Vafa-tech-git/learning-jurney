# Question 3: Card and deck implementation

This project implements a standard 52-card deck and card logic in C++.

## Features
- Definitions for card ranks and suits.
- `Deck` class to manage a collection of 52 unique cards.
- Shuffling functionality using a custom random number generator.
- Ability to deal cards from the deck.

## File structure
- `Question.cpp`: Main source code defining `Card` and `Deck` classes.
- `Random.h`: Header file for random number generation (assumed to be present in the directory).

## Usage
This file is designed to be included in other projects (like the Blackjack game in Question 4) rather than run as a standalone executable, although it may contain a main function for testing purposes.
