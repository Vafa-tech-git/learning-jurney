# Question 4: Blackjack game implementation

This project implements a simple command-line Blackjack game in C++.

## Features
- Player vs. Dealer logic.
- Basic hit and stand mechanics.
- Game state evaluation (win/lose/bust).

## File Structure
- `Question.cpp`: Main game logic and entry point.
- `Question.exe`: Compiled executable.

## How to Play
1. Compile the code using a C++ compiler (e.g., `g++ Question.cpp -o Question.exe`).
2. Run the executable.
3. Choose '(h)' to hit or '(s)' to stand when prompted.
