# Dynamic Array Name Sorting

This program asks the user how many names they want to enter, stores them in a dynamically allocated array of strings, sorts them, and prints the sorted list.

## What it does
- Prompts the user for the number of names.
- Allocates a dynamic `std::string` array.
- Collects each name from the user.
- Sorts the names alphabetically.
- Displays the sorted names.

## How to compile and run
```bash
g++ Question.cpp -o Question.exe
./Question.exe
```
