# Recursion exercises

This directory contains recursive implementations of various algorithms as part of the LearnCPP chapter on recursion.

## Problems solved

1.  **Factorial calculation**: Implementing a recursive function to find the factorial of an integer.
2.  **Sum of digits**: A recursive approach to summing every individual digit in a given number.
3.  **Decimal to binary conversion**: Using recursion to print the binary representation of a decimal number by leveraging the call stack.

## Key concepts
- **Base case**: The condition that stops the recursion.
- **Recursive step**: The part where the function calls itself with a smaller version of the problem.
- **Call stack**: Understanding how the order of operations (printing before vs. after the recursive call) affects the output.
